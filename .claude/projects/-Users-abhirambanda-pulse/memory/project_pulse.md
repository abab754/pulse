Pulse: lightweight API observability for indie devs and small teams. Core flow: create project → get API key → send telemetry → view dashboard → AI explains anomalies.

**Why:** Alternative to heavy monitoring tools (Datadog, etc.) for small teams.

**How to apply:** Keep everything simple, monolithic Next.js, Vercel-deployable. Cut scope aggressively. Demo-able with seed data. Started 2026-03-31.
